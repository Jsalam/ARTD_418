Interactive Mondrian

Recreate one geometric painting of Piet Mondrian using relative coordinates. You should submit a link to your sketch on your P5 editor. The canvas size should be proportional to the chosen painting and its shortest side must be 600 pixels. The composition scale should respond to the mouse position: the closer to the canvas origin the smaller the composition scale.

      function draw() {
        // Background
        background(0);
        noStroke();

        // Mouse scale factor
        let scale = map(mouseX, 0, width, 0.5, 1);

        // Scale gap
        gap = 16 * scale;

        // Blue rectangle
        fill(0, 0, 255);
        let blue1XStart = orgX;
        let blue1YStart = orgY;
        let blue1Xend = 252 * scale;
        let blue1Yend = 74 * scale;
        rect(blue1XStart, blue1YStart, blue1Xend, blue1Yend);

        // Red rectangle
        fill(255, 0, 0);
        let red1XStart = blue1Xend + gap;
        let red1YStart = orgY;
        let red1Xend = 314* scale;
        let red1Yend = 30 * scale;
        rect(red1XStart,red1YStart , red1Xend, red1Yend);
