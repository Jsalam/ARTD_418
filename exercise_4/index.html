Absolute and Relative Coordinates

An absolute coordinate is a coordinate that is defined in relation to the origin of the canvas. The origin of the canvas is the coordinate 0,0 located at the top left corner of the canvas. It is absolute because the X and Y values are fixed numbers that increase or decrease from 0,0. 


     // The origin of these rectangles is absolute
    
    rect(100, 100, 50, 50);
    rect(150, 100, 50, 50);
    rect(200, 100, 50, 50);

On the other hand, a relative coordinate is defined in relation to a variable point in the canvas. In other words, the origin can be displaced to any point on the canvas 


    // The width of the violet rectangles and the origin of the two on the right
    // are relative to the mouse position 
    
    let w = 50 + mouseX / 4;
    rect(100, 200, w, 50);
    rect(100 + w, 200, 50, 50);
    rect(100 + w + 50, 200, w, 50);
