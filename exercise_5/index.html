Auto-scrolling

Exercise description

This exercise is about creating a paragraph that scrolls automatically.

There are two sections where the text is displayed. The one on the top is a canvas created in p5*js. The one on the bottom is a paragraph that scrolls automatically as its content is filled with more text.

Your task is to first create the canvas in p5*js. Once it is working there you can explore the p5*js DOM reference to explore how to select elements from the HTML page and change its attributes.

The text is loaded from a .txt file with the text of your choice. You should put in your project folder. The text should be long enough to make the paragraph scroll.
